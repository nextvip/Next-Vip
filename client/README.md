# 🚀 NextVip (Client)

A React (Vite) client for **NextVip**, with a scalable folder structure and modern UI tooling.

---

## 🔧 Tech Stack

- ⚡ **Vite + React (JavaScript + SWC)**
- 💨 **Tailwind CSS**
- 🎨 **Material UI (MUI)**
- 💻 **shadcn/ui**
- 🔁 **Redux Toolkit** (fully integrated)
- 🧪 **Test API Call Setup**
- 📁 **Optimized Folder Structure**

## 🚀 Getting Started

1. **Clone the Repository**
   ```bash
   git clone <your-repo-url>
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Start Development Server**
   ```bash
   npm run dev
   ```

4. **Open your browser and navigate to:**
   http://localhost:5173
